=== WooCommerce Intuit QBMS Gateway ===
Author: woothemes, skyverge
Tags: woocommerce
Requires at least: 4.0
Tested up to: 4.4.1
Requires WooCommerce at least: 2.3.6
Tested WooCommerce up to: 2.5.0

Accept credit cards in WooCommerce with the Intuit QBMS gateway

See http://docs.woothemes.com/document/woocommerce-intuit-qbms/ for full documentation.

== Installation ==

1. Upload the entire 'woocommerce-gateway-intuit-qbms' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
