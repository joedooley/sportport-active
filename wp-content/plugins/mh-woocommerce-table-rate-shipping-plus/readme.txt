=== WooCommerce Table Rate Shipping Plus+ ===
Version: 1.5.3

== Installation ==

1. Transfer the 'mh-woocommerce-table-rate-shipping-plus' directory to the '/wp-content/plugins/' directory of your WordPress installation
2. Navigate to the WordPress admin area
3. Navigate to Plugins
4. Activate WooCommerce Table Rate Shipping Plus

If you have previously installed the free version of Table Rate Shipping, you should consider removing it to avoid confusion with the Plus version. Your settings will be automatically transferred.

== Usage ==

Documentation is available at our support website: http://mangohour.com/support/woocommerce-table-rate-shipping
