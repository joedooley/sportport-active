== Description ==
WP Smooth Scroll To Top is a easy to use and flexible WordPress plugin. This plugin place a floating back to top arrow on your site.

https://www.youtube.com/watch?v=NAvapScpA5w

== Installation ==
1. Upload `wp-smooth-scroll-to-top` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently asked questions ==
= How to change it's styles? =
This plugin allow you to manage it's styles from admin (plugins settings page).
